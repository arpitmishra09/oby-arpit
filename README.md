# Obys Agency Clone

A modern, visually appealing clone of the [Obys Agency](https://obys.agency/) website.  
This project replicates the smooth animations, creative layout, and interactive design of the original agency site.

---

## 🚀 Features
- 🎨 Pixel-perfect modern UI  
- ✨ Smooth animations & transitions (GSAP/Framer Motion/etc.)  
- 📱 Fully responsive design  
- 🌙 Dark aesthetic with creative typography  
- ⚡ Optimized for performance  

---

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 / SCSS, JavaScript (ES6+)  
- **Animations:** GSAP / Locomotive Scroll / Lenis  
- **Build Tools:** Vite / Webpack (depending on your setup)  
- **Version Control:** Git & GitHub

## 📂 Project Structure
```bash
.
├── public/           # Static assets
├── src/              # Source files
│   ├── assets/       # Images, fonts, media
│   ├── components/   # Reusable components
│   ├── styles/       # CSS/SCSS files
│   ├── scripts/      # JS files
│   └── index.html    # Main entry file
├── package.json
└── README.md


## ⚡ Getting Started
1️⃣ Clone the repository
git clone https://github.com/yourusername/obys-agency-clone.git
cd obys-agency-clone

2️⃣ Install dependencies
npm install

3️⃣ Run the development server
npm run dev

4️⃣ Build for production
npm run build

```
